# [Home Lab] Threat Hunting e Detecção de LOLBins (certutil.exe) com Wazuh e Sysmon
Laboratório prático de Threat Hunting utilizando Wazuh SIEM e Sysmon. O projeto foca na detecção de técnicas de Living off the Land (LOLBins), monitorando a execução suspeita de processos nativos do Windows como o certutil.exe e PowerShell, além da triagem de falsos positivos.
